# 🧠 기억력의 신

> 카드를 외우고 짝을 맞춰라! 스테이지가 올라갈수록 카드가 늘어납니다!

**▶ 지금 바로 플레이** → [https://mindweaver74.github.io/memory](https://mindweaver74.github.io/memory)

---

## 🎮 게임 방법

1. 스테이지 시작 시 카드가 잠깐 공개됩니다 — 위치를 외우세요!
2. 카드가 뒤집히면 짝이 맞는 카드를 찾아 클릭!
3. 제한 시간 안에 모든 짝을 맞추면 다음 스테이지로 진행
4. 시간 초과 또는 실수가 쌓이면 게임 오버

---

## 📈 스테이지 구성

스테이지가 올라갈수록 카드 수가 늘어납니다!

| 스테이지 | 카드 수 | 그리드 | 제한시간 | 힌트 노출 |
|---------|--------|--------|---------|---------|
| 1 | 4장 (2쌍) | 2×2 | 10초 | 1.6초 |
| 2 | 8장 (4쌍) | 4×2 | 15초 | 2.2초 |
| 3 | 12장 (6쌍) | 4×3 | 20초 | 2.8초 |
| 4 | 16장 (8쌍) | 4×4 | 25초 | 3.4초 |
| 5 | 20장 (10쌍) | 4×5 | 30초 | 4.0초 |
| 6 | 24장 (12쌍) | 4×6 | 35초 | 4.6초 |
| 7 | 30장 (15쌍) | 5×6 | 40초 | 5.0초 |
| 8 | 36장 (18쌍) | 6×6 | 45초 | 5.0초 |

> 카드 수에 비례해서 힌트 노출 시간도 길어집니다!

## 🔥 콤보 & 피버타임

- 연속으로 짝을 맞추면 **콤보** 보너스 점수
- 콤보가 쌓이면 **피버타임** 발동 → 점수 배율 증가!

---

## ⏱️ 점수 계산

| 항목 | 점수 |
|------|------|
| 짝 맞추기 | 기본 점수 |
| 시간 보너스 | 남은 시간 × 배율 |
| 카드 수 보너스 | 쌍 수 × 50점 |
| 스테이지 보너스 | 스테이지 × 100점 |
| 콤보 배율 | 연속 성공 시 배율 증가 |

## ✨ 특징

- 🧠 **점진적 난이도** — 4장에서 시작해 36장까지 늘어남
- ⏱️ **제한시간** — 카드 수에 맞게 조정된 시간
- 🔥 **콤보/피버** — 연속 성공으로 보너스 점수
- 📱 **모바일 최적화** — 터치 조작 완벽 지원
- 🌌 **사이버펑크 UI** — 네온 색상의 독특한 디자인
